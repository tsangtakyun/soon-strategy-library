# SOON Strategy Library

Internal strategy planning tool for SOON.

## Purpose

- manage `Budget Shapes`
- manage `Angle Types`
- manage `Deliverable Shapes`
- prepare reusable strategy logic for external campaign flows

## Current MVP

- internal login shell
- editable strategy library page
- local browser persistence

## Suggested Next Steps

1. persist library to Supabase
2. add categories / tags / status
3. expose read-only API for other SOON products
